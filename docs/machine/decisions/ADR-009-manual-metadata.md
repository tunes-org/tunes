# ADR-009: Manual metadata minimisation

- **Status:** Accepted
- **Date:** 2026-07-17

## Context

Open question #9. Heavy forms kill contribution.

## Decision

- **Required for upload:** journey/section confirmation (after alignment), consent, essential device/app provenance (automatic).
- **Optional post-recording:** carriage region (front/mid/rear), phone placement, crowding, sitting/standing.
- **Campaign-only:** richer fields for research drives.

## Consequences

Autos where reliable; never block upload on optional carriage trivia.
