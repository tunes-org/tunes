# ADR-005: Derived-only contribution default

- **Status:** Accepted
- **Date:** 2026-07-17

## Context

Open question #5. Prefer privacy-preserving uploads.

## Decision

v1 contributions upload **derived acoustic features** (and metadata/flags/optional subjective answers) only, after local review and explicit consent with field-level preview.

## Consequences

Aligns ios privacy UX and web intake schema; simplifies GDPR posture relative to raw audio hosting.
