# ADR-008: Voice Memos and imports as Tier D (optional)

- **Status:** Accepted
- **Date:** 2026-07-17

## Context

Open question #8.

## Decision

If imports are accepted at all, classify as **Tier D** (reduced control): incomplete provenance, possible voice-processing, unknown gain. Not shown as equivalent to in-app Tier B/C on the map. v1 may **omit** imports entirely until capacity exists.

## Consequences

Import UX is lower priority than native recorder quality.
