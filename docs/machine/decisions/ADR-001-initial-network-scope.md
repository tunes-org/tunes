# ADR-001: Initial network scope (London pilot)

- **Status:** Accepted (provisional for public launch breadth)
- **Date:** 2026-07-17

## Context

Open question #1: Underground only vs all London rail at the public project start.

## Decision

- **Pilot (Phase 6):** Prioritise selected Underground lines (e.g. Central, Jubilee, Victoria, Northern) for dense repeated coverage.
- **Schema:** Remain capable of Overground / Elizabeth / National Rail as network profiles.
- **Public launch breadth:** Defer hard product commitment until pilot capacity is known; do not block schema on Tube-only types.

## Consequences

Marketing copy must not promise city-wide coverage before data exist. Map honesty rules apply per section.
