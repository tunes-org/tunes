# ADR-010: Subjective survey timing (after journey)

- **Status:** Accepted
- **Date:** 2026-07-17

## Context

Open question #10.

## Decision

Collect optional perception survey **after** recording/review, not during the moving journey (safety and attention). Keep subjective fields separate from objective metrics forever.

## Consequences

ios UX sequences: align → privacy preview → optional survey → consent/upload.
