## Map honesty (near legend / first visit)

```
Survey-grade crowd data with uncertainty. Not a Class 1 engineering survey. Sample size and quality tiers matter — sparse sections are not definitive rankings.
```

## OAuth / sign-in

```
Sign in with Google to upload contributions, vote, or comment. We receive a basic identity from Google (such as an email and account ID). Hosted on Vercel; accounts stored with Supabase. See Privacy for details.
```

## Vote / comment (if enabled)

```
Be respectful. No harassment, doxxing, or medical “risk” claims. Comments may be moderated. Report problems via Report content.
```

```
Comments are reviewed before they appear in this beta.
```

## Cookie banner

```
We use necessary cookies to run this site. Optional analytics cookies help us improve TUNES. Accept optional cookies? [Reject] [Accept] · Cookie settings · Cookie notice
```

Reject and Accept must be equally prominent.

## Dataset download

```
Open releases aim to minimise personal data. Already-downloaded files cannot be recalled from the internet. Licence and version are stated per release.
```
