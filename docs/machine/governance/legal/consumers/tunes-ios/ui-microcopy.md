# UI microcopy — `tunes-ios`

**Status:** Draft — counsel review required  
**Version:** 0.1.0-draft

## Incidental speech (pre-record or consent)

```
Others’ voices may be captured in the carriage. Default upload uses derived metrics, not raw audio.
```

## Perception survey framing

```
Optional questions about how the journey felt. These are your opinions — not a medical assessment. Skip any you prefer not to answer.
```

**Do not use:** pain scales framed as diagnosis; “risk to hearing”; “unsafe exposure”.

## Child suitability (if offered — metadata only)

```
Optional: would you consider this journey suitable for a child you care for? This is your opinion about the trip — it does not enrol a child in TUNES.
```

## Quality / uncertainty

```
Phone measurements are survey-grade. Quality tier and sample context matter.
```

## Non-affiliation (about screen)

```
TUNES is independent and is not operated, approved, or endorsed by Transport for London.
```
