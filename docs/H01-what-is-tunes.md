# H01 — What is TUNES

**TUNES** = Transit User Noise Experience Survey.

## Does

- Passenger measures noise on ordinary rail journeys (iPhone-first)
- Optional short perception survey after the trip
- Publishes a quieter-journey / noise map
- Publishes an **open methodology + versioned dataset** (often the more important output)

## Does not

- Replace TfL / Class 1 professional surveys
- Imply TfL endorsement
- Claim health “risk scores” or engineering-grade accuracy at foundation stage

## Who it helps

| Audience | Value |
| --- | --- |
| Passengers / parents / sensory-sensitive travellers | Compare routes |
| Researchers / journalists / civic groups | Repeated real-world evidence |
| Operators / authorities | Recurring passenger pain points (as signals, not substitutes) |

## Principles (short)

Open · private by default · auditable · honest about uncertainty · offline-first capture · versioned · London-first, portable schema
